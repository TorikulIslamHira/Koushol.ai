# features/

Business-logic components and hooks, grouped by domain: `auth/`, `courses/`, `chapters/`, `quizzes/`, `enrollment/`. Each feature folder has its own `components/`, `hooks/`, and README — see those for what's specific to each domain.

A component goes here (not `components/ui/`) the moment it references a Supabase table, a role, a course/chapter/quiz, or any other Koushol-specific concept.
